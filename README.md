# static-site-webpack-habr

![Webpack](https://raw.githubusercontent.com/Harrix/static-site-webpack-habr/main/img/featured-image.png)

Demonstration of building HTML pages (or static site) using [Webpack](https://webpack.js.org/) and [Node.js](https://nodejs.org/).

Article (contains outdated version `v1.0`): <https://habr.com/ru/articles/350886/> (see release: <https://github.com/Harrix/static-site-webpack-habr/releases/tag/v1.0>).

New article draft: [`docs/article_new.md`](docs/article_new.md).

## 🚀 Getting started

### ✅ Prerequisites

- [Node.js](https://nodejs.org/) (npm comes with Node.js)

### 📦 Installation

```bash
npm install
```

### 🧑‍💻 Development

Start the dev server:

```bash
npm start
```

Or run development builds:

```bash
npm run dev
```

Watch mode:

```bash
npm run watch
```

### 🏗️ Build

Create a production build:

```bash
npm run build
```

### 🧰 Other scripts

| Command         | Description                          |
| --------------- | ------------------------------------ |
| `npm start`     | Dev server                           |
| `npm run dev`   | One-off development build            |
| `npm run watch` | Development build with file watching |
| `npm run build` | Production build                     |
